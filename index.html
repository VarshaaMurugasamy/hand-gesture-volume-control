About this demo

This is a browser-based version of the original desktop project, which controls your real Windows system volume using Python, OpenCV, MediaPipe, and Pycaw. Browsers can't access system audio directly for security reasons, so this version controls an on-screen virtual volume bar instead — but the hand-tracking and gesture logic are the same.
